-=[ Arjunane-datepicker.js v1 DEMO ]=-

Dependency-free Javascript library Date Picker v1
Required:
  • arjunane-date.css
  • arjunane-date.js
Simple init 
kalender.dom('.date').ex({
    days: 
    {
        'Minggu':'Min',
        'Senin':'Sen',
        'Selasa':'Sel',
        'Rabu':'Rab',
        'Kamis':'Kam',
        "Jum'at":'Jum',
        'Sabtu':'Sab'
    },
    format: 'yyyy-mm-dd',
    time: true,
    year_range: '1995:2020' // OR year_range: "2020"
});
