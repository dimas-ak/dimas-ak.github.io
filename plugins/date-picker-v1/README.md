# date-picker-v1
Date Picker v1

How do it used?
```javascript
 kalender.dom('.date').ex({
            days: {
                    'Minggu':'Min',
                    'Senin':'Sen',
                    'Selasa':'Sel',
                    'Rabu':'Rab',
                    'Kamis':'Kam',
                    "Jum'at":'Jum',
                    'Sabtu':'Sab'
                  },
            format: 'yyyy-mm-ddd',
            time: true,
            year_range: '1995:2020'
        });
```
